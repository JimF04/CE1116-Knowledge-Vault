## 📚 Idea/Concepto 
Modelo de computación en la nube donde un proveedor externo ofrece recursos como servidores, almacenamiento y red, compartidos entre múltiples clientes mediante un esquema multi-tenant. El usuario paga únicamente por los recursos que consume, mientras que el proveedor administra la infraestructura física bajo un modelo de responsabilidad compartida cuyo alcance varía entre IaaS, PaaS y SaaS. Además, el cloud público opera sobre una infraestructura global compuesta por Regiones, Zonas de Disponibilidad y una red troncal de alta capacidad, permitiendo autoservicio bajo demanda, aprovisionamiento mediante APIs y escalabilidad rápida para adaptarse a las necesidades del cliente.

## 📌 Puntos Claves 
- Infraestructura compartida (multi-tenant)
- Pago por uso
- Base de todos los servicios cloud
- Soporta regiones y zonas
- Permite escalabilidad
