## 📚 Idea/Concepto 

Una Virtual Private Cloud (VPC) es una red virtual aislada dentro de un Cloud Público, creada por completo mediante software y basada en redes definidas por software (SDN). Proporciona al usuario control sobre direcciones IP, subredes, tablas de ruteo y firewalls, mientras el proveedor se encarga del hardware físico. La VPC separa el plano de control y el plano de datos para optimizar el enrutamiento virtual y garantizar aislamiento de rendimiento en entornos multi-tenant. Además, puede integrarse con redes locales mediante VPN o enlaces dedicados para soportar arquitecturas híbridas. 
## 📌 Puntos Claves 
-  Red aislada dentro del cloud público
- Basada en SDN
- Control de red lógico (IP, routing, firewalls)
- Integración con redes locales
